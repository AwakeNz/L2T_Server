Server Commands

.disablenicknamewings
With this command you can disable their nickname wings.

.disablearmorglow
With this command you can disable YOUR armor enchant glow visual effect.

.sell
With this command you can set up custom private stores.

.treasure
Special event command for check the location from the Hidden Treasures.

.itemid
With this command you can see an unique item id which can help you in case of steal.

.noexp
With this command you can turn off the xp gain.

.landrates
With this command you can see debuff land rates.

.refusebuff
With this command you can refuse the incoming buffs.

.blockrequests
With this command you can refuse the incoming requests.

.blockpms
With this command you can refuse the incoming pms.

.refusekillinfo
With this command you can refuse the kill info pop up page.

.stabs
With this command you can see the stabs angle.

.unrec
With this command you can delete all your rec points.

.time
With this command you can check the current hour on server.

.gotolove
Only for married couples. Transports you to the exact point where is your partner.

.event
This command will pop-up the Community Board window with the current event info, their participants and your state.
